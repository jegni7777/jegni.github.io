rstudio.com에서 R studio 무료 버젼을 다운로드 받습니다.

cheetsheets를 이 곳에서 다운로드 받아 사용에 참고합니다.
업데이트 메일링 서비스를 이용하면 좋을 것 같습니다.


?command : 명령어의 안내 문서를 호출합니다.
str() : 오브젝트의 정보를 호출합니다.
class : 오브젝트의 타입을 호출합니다.
library() : 패키지를 사용가능 하도록 불러옵니다.
install.packages() : 패키지를 설치합니다.
getwd() : 현재 작업 디렉토리(working directory)를 문자열(chr) 타입으로 호출합니다.
setwd() : 현재 작업 디렉토리를 변경합니다. chr 타입의 path를 통과시킵니다.
read.csv("path/name.csv") : csv 파일을 호출합니다.
e.g. LSD <- read.csv("/Users/jaegeunkim/projectR/LSdata10.csv")
실행시 오류가 발생하여 사용 불가
read.table 함수를 대신 사용합니다.

read.table() : read.csv()보다 범용성이 높습니다.
e.g. read.table("path", sep = ",", fileEncoding = "euc-kr")
separator를 콤마로 설정해야 합니다
맥과 윈도우 사이에 인코딩 방식 차이가 존재하므로, 필요한 경우에 이를 지정해주어야 한다.

View() : dataframe 또는 tibble 오브젝트를 윈도우에 출력합니다. head() : view()와 유사하지만, 처음 6행만 콘솔에 출력합니다.
R은 대소문자를 구별합니다. view()로 쓰지 않습니다.
